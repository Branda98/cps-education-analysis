# Data Instructions

The dataset used in this project comes from the Current Population Survey (CPS), provided by the U.S. Census Bureau and Bureau of Labor Statistics.

Due to file size limitations, the dataset is not included in this repository.

## How to Access the Data

1. Download the CPS dataset from:
   https://www.census.gov/data/datasets/time-series/demo/cps/cps-basic.2022.html#form-dropdown-1979780401
